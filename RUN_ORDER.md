# EAHE MATLAB Run Order

Recommended execution order in MATLAB:

For final clean generation of all outputs:

```matlab
generate_all_outputs
check_result_revision
audit_result_physics
```

If only a small number of files are reported as old or missing:

```matlab
rerun_stale_results
```

If MATLAB cannot start but result `.mat/.csv` files already exist, use the Python fallback exporter:

```powershell
python .\python_export_available_outputs.py
```

For paper-style selected figures and analytical-model comparison:

```powershell
python .\paper_postprocess.py
python .\ozgener2011_validation_postprocess.py
```

For core paper results:

```matlab
main_core_results
```

For all results including numerical validation:

```matlab
main_all_results
finalize_results_after_run
```

For step-by-step execution:

```matlab
main_baseline
main_check_undisturbed_soil_phase
main_temperature_field_fine
main_degradation_validation
main_sensitivity_gap
main_sensitivity_gap_conductivity
main_sensitivity_operation
main_sensitivity_soil_layer2
main_variable_gap_extension
main_sensitivity_variable_gap
main_numerical_validation
main_compare_homogeneous_degradation
main_ozgener2011_validation
check_result_revision
audit_result_physics
regenerate_baseline_figures
regenerate_variable_gap_sensitivity_figures
check_generated_results
collect_summary_tables
extract_key_findings
finalize_results_after_run
```

Primary outputs:

- `baseline_result.mat`: baseline transient result.
- `degradation_validation_result.mat`: no-gap vs fixed-gap 30-day degradation comparison.
- `gap_sensitivity_result.mat`: gap-thickness sensitivity.
- `gap_conductivity_sensitivity_result.mat`: gap effective conductivity sensitivity.
- `operation_sensitivity_result.mat`: mass-flow-rate and pipe-length sensitivity.
- `soil_layer2_sensitivity_result.mat`: second-layer soil conductivity sensitivity.
- `temperature_field_fine_result.mat`: high-resolution temperature contour source data.
- `variable_gap_extension_result.mat`: fixed-gap vs temperature-dependent gap-resistance extension.
- `variable_gap_sensitivity_result.mat`: sensitivity to the temperature coefficient in the variable gap-resistance model.
- `numerical_validation_result.mat`: time-step and grid independence checks.
- `figures/`: PNG and FIG exports generated by `main_core_results` or `save_all_open_figures`.
- `EAHE_summary_tables.xlsx`: collected CSV summaries in one workbook.
- `EAHE_key_findings.txt`: compact text report with key numerical findings.
- `EAHE_results_image_review.md`: image-level reasonableness review and figure-fix notes.
- `EAHE_code_logic_audit.md`: code-level logic audit and consistency fixes.
- `audit_result_physics.m`: quick physical-consistency checks for phase alignment, cooling sign, degradation trend, and contour source data.
- `rerun_stale_results.m`: selective rerun script for stale operation, numerical-validation, and variable-gap comparison outputs.
- `paper_postprocess.py`: paper-style figure selection, figure beautification, and analytical-model comparison.
- `paper_validation_workflow.md`: validation workflow for homogeneous-soil, experimental, multi-experiment, and analytical comparisons.
- `main_compare_homogeneous_degradation.m`: layered-soil versus homogeneous-soil degradation comparison.
- `main_experimental_comparison.m`: experimental-data comparison after digitized data are filled.
- `main_ozgener2011_validation.m`: reduced homogeneous no-gap validation case using Ozgener et al. (2011) experimental parameters.
- `ozgener2011_validation_postprocess.py`: post-process digitized Ozgener heat-resistance data and model validation output.
- `literature_experiment_manifest.csv`: candidate experimental-validation study list.
- `experimental_comparison_timeseries.csv`: template for digitized experimental time series.
- `EAHE_RC_paper_draft.md`: paper draft with numbered mathematical equations.

For paper figures:

- Use `plot_eahe_summary(result)` for baseline transient and degradation figures.
- Use `main_check_undisturbed_soil_phase` to verify that the EAHE operation starts from the intended summer soil-temperature phase.
- Use `plot_temperature_contour(result, snapshotIndex)` for near-pipe temperature contours.
- Use `main_temperature_field_fine` for paper-quality contours (`Ntheta = 16`, `Nr = 10`).
- Use `finalize_results_after_run` after long simulations finish to refresh fixed figures and collect summary outputs.
- Use `generate_all_outputs` to delete old PNG/FIG files, rerun all analyses, regenerate figures, and export summary data.
- For heat transfer rate plots, the plotting function skips the initial `Q=0` point to avoid a nonphysical vertical display artifact.
